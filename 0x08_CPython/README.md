# 0x08 CPython

Exploring CPython internals using C extensions.

## Tasks

- **0-python_lists_bigO**: Big O notation for Python list operations.
- **1-python.c**: Print info about a Python list (`print_python_list`).
- **2-python.c**: Print info about Python lists and bytes objects.
- **3-python.c**: Print info about Python lists, bytes, and float objects.
- **4-python.c**: Print info about Python string objects (Unicode internals).
- **5-python.c**: Print Python integers using only `PyLong_Check`/`PyLong_SHIFT`.
